## 一、配色板（按像素占比量化）

主背景是 **绿 → 青** 的对角渐变，叠半调圆点，深蓝navy 做描边/楼宇/文字。

| 角色 | Hex | 占比 | 说明 |
|------|-----|------|------|
| 深蓝 navy | `#10203A` | 11.9% | 标题描边、楼宇剪影、深色文字 |
| 春绿 mint | `#7CFAA2` | 11.9% | 背景渐变上半 |
| 奶白 cream | `#F4FCEC` | 10.4% | 半调圆点 / 高光留白 |
| 青绿松石 turquoise | `#2DDBDD` | 8.5% | 背景渐变中段 |
| 薄荷绿 teal-green | `#62EFB5` | 8.1% | 渐变过渡 |
| 嫩绿 | `#6FF5AB` | 7.3% | 渐变过渡 |
| 暖橙(人物外套) | `#D06052` | 7.0% | 滑板少年外套（混阴影后的量化值，橙系） |
| 水绿 aqua | `#47E5C9` | 6.5% | 渐变过渡 |
| 宝蓝 royal blue | `#395DAA` | 6.2% | 楼宇/鞋面点缀 |
| 淡黄 pale yellow | `#FAF78E` | 5.0% | 星星、纸飞机、日期框底 |
| 天蓝 sky cyan | `#17C4E8` | 4.3% | 渐变下半 |
| 粉肤 skin/pink | `#DFBCC5` | 3.8% | 人物肤色 |
| 黑描边 | `#000001` | 0.4% | 人物/图标硬描边 |

**定点补采**（背景四角，确认渐变走向）：
- 左上背景绿 `#82FC9C` → 右下/底部青蓝 `#06CCFA` / `#55A3D6`
- 日期框亮黄 `#FDF885`
- 人物头发紫（小面积，约 `#7E5A90`，量化未单列）

### 标题主视觉配色（从独立标题 PNG 精采，992×470）

贴纸描边标题字 `assets/brisbane-2026-s1-title.png`，是整套视觉里最强的识别元素：

| 角色 | Hex | 说明 |
|------|-----|------|
| 标题描边 navy | `#051720` | 近黑深蓝，粗描边（比封面背景 navy `#10203A` 更深） |
| 白字填充 | `#FFFFFF` / `#ECFDFE` | 「布里斯班三校」「UQ/QUT/GRIFFITH」主行，略带冷调白 |
| 亮青字 | `#5CFCFF` | 「2026 S1」「三校新生节」点睛行 |
| 柠檬绿爆发线 | `#D8FF78` | 标题两侧速度/爆发线 |

> 配色逻辑：navy 粗描边兜底 → 白字主信息 → 亮青字 `#5CFCFF` + 柠檬绿线 `#D8FF78` 做撞色点睛。和背景的绿青渐变同色系呼应。

### 主视觉插画配色（从独立人物 PNG 精采，1020×1277）

滑板少年插画 `assets/brisbane-2026-s1-character.png`，3D 厚涂、动态跳跃姿势：

| 部位 | 主色 | 高光 / 阴影 |
|------|------|------------|
| 外套（橙） | `#E76141` | 高光 `#EA7A5E` · `#F18D7A` / 阴影 `#BD4B4C` |
| 头发（紫） | `#5D4370` | — |
| 裤子（蓝） | `#1D3265` | 深 `#132656` / 中 `#224A89` / 高光 `#4B82BF` |
| 肤色（粉） | `#F3BCD4` | 高光 `#F7DCD4` |
| 鞋 / 内搭衫（浅蓝） | `#9AC6E3` | 鞋面亮蓝 `#4B82BF`，鞋底橙线呼应外套 |

> 人物用**橙 vs 蓝**强对比（暖外套 + 冷裤子），紫发做第三色点缀；和背景绿青渐变形成「冷底 + 暖主角」的视觉聚焦。

### 简化版（可直接拿去做 token）
```
--green-mint:   #7CFAA2   /* 渐变起点 */

--cream:        #F4FCEC   /* 半调点/留白 */

--accent-yellow:#FAF78E   /* 黄色点缀 */

/* 标题主视觉 */
--title-stroke: #051720   /* navy 粗描边 */
--title-fill:   #FFFFFF   /* 白字主信息 */
--title-cyan:   #5CFCFF   /* 亮青点睛字 */
--title-lime:   #D8FF78   /* 柠檬绿爆发线 */
```

## 二、设计元素清单

| 元素 | 描述 |
|------|------|
| 主视觉 | 橙色外套、紫发的青年「滑板/向前冲」插画，3D 厚涂风 |
| 背景 | 绿→青对角渐变 + 半调圆点（halftone）纹理 |
| 标题字 | 白色填充 + navy 粗描边（贴纸感），"2026 S1" 用青色字 |
| 装饰小物 | 黄色星星、黄色纸飞机、绿色书本、navy 城市楼宇剪影（橙色窗格） |
| 强调笔触 | 标题两侧的爆发线 / 速度线 |
| 日期框 | 淡黄圆角块 + 青色描边 + navy 文字 |
| Logo | 匠人学院 JR ACADEMY（黑色版，左上角） |

## 三、风格关键词
青春 / 活力 / 校园 / 3D 厚涂插画 / 贴纸描边字 / 半调圆点 / 高饱和撞色（绿青 base + 橙黄点缀）。

## ⚠️ 能力边界（Canva MCP）

- ✅ 可导出**整页** PNG/JPG/PDF/PPTX（已导出封面页存 `assets/`）
- ❌ **不能逐个拆分下载**单个矢量元素（人物/图标/星星等）。Canva MCP 无 per-element 资产导出接口。要单独拿某个元素，只能：(a) 整页导出后人工裁切；(b) 回 Canva 里选中元素右键下载；(c) 让设计同学从源工程导出 PNG/SVG。
- ✅ 配色可按上表精确复刻（已量化）。
